Shirley's Temperature Chamber

Welcome to Shirley's Temperature Chamber! This interactive website allows you to set and change the target and current temperatures, and view the temperature changes in real-time on a dynamic chart. Input numbers are limited to a range of -20 to 200.

📝Setting the Target Temperature

Enter the desired value (between -20 and 200) in the "Set Temperature" input field and select the unit from the dropdown menu (Celsius, Kelvin, or Fahrenheit).

🌡Adjusting the Current Temperature

Enter the desired value (between -20 and 200) in the "Current Temperature" input field. The value will automatically update as it gradually changes to match the target temperature. You can also change the unit in the dropdown menu.

📊Viewing the Temperature Chart

Visualize the current temperature changes as they happen in real time on an interactive chart that records and displays the data. Press "Start Recording" to begin tracking temperature changes over time, and press "Stop Recording" to pause the tracking.
